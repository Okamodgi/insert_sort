# insert_sort

# ОПИСАНИЕ :

Программа реализует алгоритм сортировки вставками для массивов целых чисел. Поддерживает два направления сортировки:

- По возрастанию
- По убыванию

Данные могут быть получены двумя способами:
1. Загрузка из текстового файла
2. Генерация случайных чисел в заданном диапазоне

Результаты сортировки сохраняются в файл и выводятся на экран вместе со статистикой выполнения.

# ПРИНЦИП РАБОТЫ СОРТИРОВКИ ВСТАВКАМИ

Алгоритм последовательно проходит по массиву, извлекая каждый элемент и вставляя его в правильную позицию в уже отсортированной части массива

# ФОРМАТ ВХОДНЫХ/ВЫХОДНЫХ ФАЙЛОВ

Файлы содержат последовательность целых чисел

Числа разделяются пробелами и/или переводами строк

Пример:
42 17 8 93 11 56 3 27

# ОСНОВНЫЕ ФУНКЦИИ

**insertionSort()** - Основная функция сортировки вставками
**generateRandomArray()** - Генерация массива случайных чисел
**readFromFile()** -	Чтение массива из файла
**writeToFile()** - Запись массива в файл

# ОБРАБОТКА ОШИБОК
Программа проверяет:
- Корректность ввода пользователя;
- Существование входного файла;
- Возможность записи в выходной файл;
- Выделение памяти для массива.

При возникновении ошибки выводится соответствующее сообщение, и программа запрашивает повторный ввод
